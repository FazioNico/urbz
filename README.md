# Urbz Template
Project for Urbz

## Dev Start
- `$ nvm use 6` || `$ node 6`
- `$ npm install`
- run `MAMP` || `WAMP` server from `./dist` folder
- open project in good ide and run `$ gulp`

## Prod Start
- run mamp server from `./dist` folder
- open from `./index.php` file

## Sources
All sources available in dev folder.
